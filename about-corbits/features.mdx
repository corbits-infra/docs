[**Corbits**](intro) together with [**Faremeter**](faremeter) empower developers and enterprises to build the next generation of agentic, payment-enabled APIs with rapid integration and global reach. Together, they form a foundation for the autonomous internet, enabling seamless, instant payments and next-gen API monetization for both humans and AI agents.

## Best-in-class facilitation

Corbits is a production-ready facilitator any AI Agent can use built by ABK Labs. Corbits excels by leveraging the Faremeter framework which is open-source, modular, and purpose built for x402.

- **Award winning infrastructure:** Corbits utilizes industry-leading bare metal partners for RPC services.
- **No bloat:** Every line of code with Corbits is purpose-typed to ensure the fastest performance.
- **Fastest settlement**: Corbits is equipped with best in class networks and protocols for settlement.
